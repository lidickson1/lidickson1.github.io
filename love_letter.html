Love Letter

A full stack multiplayer web browser game based on the card game Love Letterâ„¢. Built using React, NextUI, Express, and Socket.IO. Deployed on Vercel and Fly.io. The game can be played both desktop and mobile devices. It also has full bot support and custom lobbies.

User Interface

The user interface has been developed to be user friendly through multiple iterations based on player feedback. A tutorial is included for new players to learn how to play the game. Players can create their own rooms or join other people's rooms to play. The website natively supports light and dark theme, and is playable both on desktop and mobile. It can even be installed as a progressive web app on mobile devices which comes with notifications support.

Bot Support

The room's host can choose to add as many bots as they want if they want to have more players. These bots will play cards like a normal player. Therefore, you can play this game as singleplayer, making it a perfect app to kill time when you're bored.
